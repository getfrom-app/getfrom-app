Find any note in seconds: From's filter system

A system that captures everything but lets you find nothing is useless. From has three levels of filters so anything you've captured is less than 5 seconds away.

The three levels of the filter system

Retrieval is as important as capture. A system where you store everything but can't find anything has the same practical value as storing nothing. That's why From has three levels of filters, each for a different use case.

The goal is that anything you've captured is less than 5 seconds away. Without remembering which folder it's in, without technical search operators, without manually navigating the tree.

Level 1: one-click chips

Chips are predefined filter buttons that appear when you open the search bar (Cmd+F). They're the most commonly used filters, available with a single click:

  • Today — Everything under today's note
  • Tomorrow — Everything under tomorrow's note
  • This week — Everything in the next 7 days
  • Overdue — Tasks with a past date that aren't complete
  • Pending — All unchecked tasks
  • Done — Completed tasks
  • Task — Only nodes marked as tasks
  • Event — Only calendar events

You can combine chips: "Pending" + "This week" shows pending tasks for the next 7 days. "Overdue" + "Task" shows only overdue tasks, excluding notes.

Level 2: text and natural language

If chips aren't enough, you can type directly in the search bar. From accepts two types of text input:

Text search

Type any word and From filters all nodes that contain it. Search is real-time — results appear as you type.

Natural language with AI

Write a query in natural language and the AI interprets it. "Meeting notes from last week", "tasks for the web project", "everything related to client García". The AI translates the intent into technical filters and shows the results.

Level 3: saved panels

Filters you use more than once can be saved as panels in the left sidebar. One click and the filter applies immediately — no need to open the search bar or select chips.

From ships with two panels: "Pending" and "This week". You can add your own from the save panel menu on any active filter.

Panels users most commonly add:

  • "Overdue" — To review what's running behind
  • "Learning this week" — To filter recent learning notes
  • "Unprocessed ideas" — To review ideas pending evaluation
  • "This week minus today's tasks" — To plan the rest of the week

💡 Advanced tip: The "Task" shortcut in the sidebar acts as your universal task inbox — it shows all tasks in the system, regardless of where they live. Perfect for the weekly review.

Most useful filter combinations

These are the combinations that deliver the most value day to day:

  • Pending + Today — The tasks you need to do today. Your daily list.
  • Overdue + Task — Tasks running behind. Urgency review.
  • This week + Event — All your commitments for the week. Agenda view.
  • Pending + This week — Your plan for the week. Foundation for Monday planning.
  • Done + Today — What you completed today. For end-of-day review.

Results with context: the tree as reference

Unlike the flat search in other apps, results in From always show the complete hierarchical context. If a task is nested under "Web project > Design phase > Tuesday meeting", you see that hierarchy in the result.

This eliminates ambiguity: when two notes share the same name in different projects, the context tells you which is which without having to open them.
