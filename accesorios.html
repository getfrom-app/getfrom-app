Arsenal

Captura en From desde cualquier sitio

Da igual dónde estés trabajando: manda lo que tengas a tu nota de hoy con una tecla, un comando o un clic. From entiende lo que guardas —tipo, fecha, contexto— y lo coloca en su sitio.

Atajo de Apple

Una tecla global desde cualquier app del Mac. Pulsa, escribe y cae en tu nota de hoy — sin abrir nada.

Crea un Atajo con la acción «Abrir URL»: from://capture?text=[Texto]&silent=1, y asígnale la tecla que quieras. Guía completa en el Manual.

Ver guía →

Raycast

Crea, busca y abre tu nota de hoy desde Raycast. La inteligencia de From decide si es nota, tarea o evento.

Instala la extensión From desde la Raycast Store y pega tu token de API (Ajustes → Accesorios → Raycast).

Raycast Store →

Chrome

Clic en el icono y guarda la página actual en tu nota de hoy. Selecciona texto y mándalo como nodo con el menú contextual.

Instala la extensión From desde la Chrome Web Store y pega tu token de API en sus opciones (Ajustes → Accesorios → Chrome).

Chrome Web Store →

Todos usan el mismo token de API de tu cuenta (Ajustes → Accesorios). Lo generas una vez y sirve para Raycast, Chrome y la extensión de Claude.
