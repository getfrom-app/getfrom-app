Arsenal

Captura en Fromly desde cualquier sitio

Da igual dónde estés trabajando: manda lo que tengas a tu nota de hoy con una tecla, un comando o un clic. Fromly entiende lo que guardas —tipo, fecha, contexto— y lo coloca en su sitio.

Atajo de Apple

Una tecla global desde cualquier app del Mac. Pulsa, escribe y cae en tu nota de hoy — sin abrir nada.

Instálalo en un clic: te pide el texto y lo manda a tu nota de hoy. Luego asígnale la tecla global que quieras desde la app Atajos.

↓ Instalar atajo de Apple →

Raycast

Crea, busca y abre tu nota de hoy desde Raycast. La inteligencia de Fromly decide si es nota, tarea o evento.

Instala la extensión Fromly desde la Raycast Store y pega tu token de API (Ajustes → Accesorios → Raycast).

Raycast Store →

Chrome

Clic en el icono y guarda la página actual en tu nota de hoy. Selecciona texto y mándalo como nodo con el menú contextual.

Instala la extensión Fromly desde la Chrome Web Store y pega tu token de API en sus opciones (Ajustes → Accesorios → Chrome).

Chrome Web Store →

Claude (MCP)

Conecta Claude con tu vault de Fromly. Guarda notas desde una conversación, crea tareas y consulta tu información directamente en Claude Desktop, Claude.ai o Claude Code — sin salir del chat.

Abre Claude → Conectores → busca Fromly. Sin tokens ni configuración manual.

Ver instrucciones de conexión →

Los accesorios de captura (Raycast, Chrome, Atajo) usan el mismo token de API de tu cuenta (Ajustes → Accesorios). Lo generas una vez.
