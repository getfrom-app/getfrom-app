Instalar en 3 pasos

Usando la app de Claude Desktop en Mac

1

Descarga From.dxt

Descarga el archivo .dxt y haz doble clic. Claude Desktop lo instala automáticamente y te pide el token.

2

Genera y pega tu token

En From: Ajustes → Integraciones → Generar token. From te lo muestra listo para copiar. Pégalo en Claude Desktop cuando te lo pida.

3

Activa el guardado automático

From te da un bloque de texto para pegar en Claude Desktop → Ajustes → Perfil → Instrucciones personalizadas. Una vez hecho, Claude guarda todo automáticamente.

¿Usas Claude Code?

Si trabajas con Claude Code (la CLI), añade manualmente la configuración a ~/.claude.json bajo la clave mcpServers:

"mcpServers": { "from": { "type": "http", "url": "https://from-server-production.up.railway.app/mcp", "headers": { "Authorization": "Bearer TU_TOKEN" } } }

Reinicia Claude Code. From ya guardará automáticamente usando el prompt de instrucciones personalizadas.

Lo que Claude puede hacer con From

Herramientas que Claude usa automáticamente según lo que le pidas.

from_setup

Activar protocolo

Genera las instrucciones para activar el guardado automático en todas las conversaciones.

from_get_today_note

Nota del día

Obtiene la nota del diario de hoy. Claude la usa como destino para guardar conversaciones y artefactos.

from_get_context

Cargar @contexto

Lee el contenido de un @contexto de área (ej: "la-isla", "inversion"). Claude lo carga al inicio de cada sesión relevante.

from_create_node

Crear nota o tarea

Crea notas en markdown o tareas con fecha de vencimiento. Cuando dices "fin", guarda el resumen de la sesión.

from_update_node

Editar contenido

Modifica texto, cuerpo o estado de cualquier nodo. Marca tareas como hechas desde el chat.

from_search

Buscar en tu vault

Encuentra notas y tareas por texto. Claude busca antes de responder para darte contexto real.

from_list_nodes

Listar tareas y notas

Consulta tareas pendientes o notas recientes con filtros. "¿Qué tengo pendiente esta semana?"

from_update_context

Actualizar @contexto

Al cerrar sesión, Claude actualiza el @contexto del área con la información nueva más relevante.

✓
"Marca como hecha la tarea de llamar a Adrián" — Claude la actualiza en tu vault al instante.

Requisitos

Necesitas From (Mac, iPhone o web) con una cuenta activa y al menos una sincronización realizada.
Compatible con Claude Desktop (instalación con .dxt, recomendada) y Claude Code (configuración manual).

El token de API se genera desde la app y dura 1 año.
Todos los datos se guardan en tu vault de From.
